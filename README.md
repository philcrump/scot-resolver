# scot-resolver

Resolver Interface for the SCOT.

* 22-32V DC Input
* STM32F042K6 Microcontroller @ 48MHz
* MCP2562 CAN 2.0A/B Communication Interface
* AD2S1210 Resolver to Digital Converter
* AD8397 Excitation Driver
* AD8692 Receiver Amplifiers
